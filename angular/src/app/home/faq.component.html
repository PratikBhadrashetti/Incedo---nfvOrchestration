General questions

How to list all catalogs?

Click on CATALOG option in the Navigation menu, select List and view all catalogs

How to add a catalog?

Click on CATALOG option in the Navigation menu, select Add and fill all the fields then click Save

How to modify a catalog?

Click on CATALOG option in the Navigation menu, select Modify and click on icon, modify the fields required then click Save

How to associate a VNF to catalog?

Click on CATALOG option in the Navigation menu, select List and click on icon, then click Associate. select VNF's and Save

How to dissociate a VNF from catalog?

Click on CATALOG option in the Navigation menu, select List and click on icon, select VNF's and click dissociate

How to launch a VNF?

Click on VNF option in the Navigation menu, select List and click Launch

How to view a launched VNF?

Click on VNF option in the Navigation menu, select Instances and view all launched VNF

How to add users to launched VNF?

Click on VNF option in the Navigation menu, select Instances, click Manage, select users and Save

How to delete a launched VNF?

Click on VNF option in the Navigation menu, select Instances and view all launched VNF . If status is 'ACTIVE' then user can click Delete

How to check the launched VNF status?

Click on STATUS option in the Navigation menu, and view all launched VNF status

How to launch ELK?

Click on STATUS option in the Navigation menu, and view all launched VNF . If status is 'ACTIVE' then user can click Launch ELK
